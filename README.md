# Pagewright

**Conversion-engineered landing page templates, with the copy already written.**

Most landing page templates hand you a pretty empty shell and leave the two hardest jobs to you: the section order that converts, and the words that fill it. Pagewright does both, and explains why, so a founder who is not a designer or a copywriter can ship a credible, converting page in about an hour.

Plain HTML and precompiled Tailwind. No framework, no build step. Open a file, change the words, change one color variable, deploy anywhere static files run.

## Free: Pagewright Lite

The [`/lite`](./lite) folder is a complete, free landing page template. Use it for anything, personal or commercial, just keep the small "Built with Pagewright" credit in the footer. See [`/lite/LICENSE`](./lite/LICENSE).

```bash
# try it in 10 seconds
git clone https://github.com/leonbede7/pagewright
cd pagewright/lite
# open index.html in your browser, then edit the words
```
